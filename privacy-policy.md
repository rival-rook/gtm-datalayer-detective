# Privacy Policy for GTM DataLayer Detective

**Last Updated:** July 13, 2026

## Data Collection

GTM DataLayer Detective does **NOT** collect, store, or transmit any personal data or browsing information to external servers.

## Data Usage

The extension operates entirely within your browser and only:

- Intercepts dataLayer events on pages you visit (for display purposes only)
- Monitors network requests to Google Analytics endpoints
- Stores data temporarily in your browser's memory while DevTools is open

## Data Storage

All captured events are stored temporarily in your browser's memory and are cleared when you:

- Close the DevTools panel
- Click the "Clear" button
- Navigate away from the page (unless "Preserve log" is enabled)

## Host Permissions

The extension requests access to all websites (`<all_urls>`) to:

- Intercept Google Tag Manager dataLayer events on any website you visit
- Monitor GA4 network requests

This access is used **ONLY** for debugging purposes within DevTools. No data leaves your browser.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking.
