# GTM DataLayer Detective Support

Welcome to the support page for GTM DataLayer Detective.

## About

GTM DataLayer Detective is a Chrome DevTools extension that helps analysts, developers, and marketers inspect:

- Google Tag Manager dataLayer events
- GA4 network requests
- Event parameters and payloads
- Real-time client-side tracking activity

## Common Questions

### I don't see any events

Make sure:

- Chrome DevTools is open
- The GTM DataLayer Detective panel is selected
- The page is generating dataLayer activity
- You refresh the page after opening DevTools

### Why does the extension require access to all websites?

The extension must be able to inspect dataLayer activity and GA4 requests on any website you choose to debug.

All processing occurs locally in your browser. No data is transmitted to external servers.

### Does the extension collect data?

No.

GTM DataLayer Detective does not collect, store, or transmit browsing activity, analytics data, or personal information.

See the Privacy Policy for details.
